# Campus Bazaar Landing Page

This repository contains the code for the landing page for Campus Bazaar, a student marketplace platform. The landing page is built using React and Tailwind CSS, providing a responsive and user-friendly interface for visitors.

## Try it Live

You can view the live version of the Campus Bazaar landing page at https://campus-bazaar-marketplace.netlify.app

## How to Run Locally

1. Clone the repository:
    ```bash
    git clone https://github.com/your-username/campus-bazaar.git
    ```
2. Navigate to the project directory:
    ```bash
    cd campus-bazaar
    ```
3. Install the dependencies:
    ```bash
    npm install
    ```
4. Start the development server:
    ```bash
    npm start
    ```
5. Open your browser and go to `http://localhost:3000` to view the landing page.
